# 계산서 출력 프로그램

1. 총 몇개의 물품을 구입했는지 물어보자.
2. 각 물품 별로, 이름, 가격, 수량을 입력받자
3. 결과를 아래와 같은 형태로 출력하자.
```
    --------------------------------------
     3 items
    --------------------------------------
      - AAA	$ 234	*	10	=	$ 2340
      - BBB	$ 1234	*	2	=	$ 2468
      - CCC	$ 340	*	3	=	$ 1020
    --------------------------------------
     Total: $ 5828
    --------------------------------------
```
